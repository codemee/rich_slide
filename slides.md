# 跟外界溝通的兩個問題

```

       o             +---------+
      / \            | claude  | 無法獲取外部資訊
     -----        +--+ ChatGPT | 無法執行實際動作
    /     \       |  |   ...   | 無止盡的複製貼上
   ---------      |  +---------+    
__/         \__   |
 |_    []   _|🕻---+  +---------+
 |_   /||\  _|       |         | 先問 LLM 要做什麼 
 |_    /\   _|🕻------+ LLM API | 再把指示轉換成動作
---------------      |         | 複雜而且具不穩定性
///////////////      +---------+

```

# 但這有兩個問題

```

       o             +---------+      遠端伺服器（碰不到使用者的電腦）
      / \            |         |     +---------+
     -----        +--+ ChatGPT +🕻----+ actions +-----> 規格不同
    /     \       |  |         |     +---------+       無法互通
   ---------      |  +---------+                          ^
__/         \__   |          本機程式內的函式（無法分享） |

 |_    /\   _|🕻------+ OpenAI  | Function calling | +-----+
---------------      | API     +------------------+ |
```

# claude 制訂 MCP 想要一統江湖

```
                       MCP Host（與 LLM 溝通）                  遠端/本機皆可
       o             +-----------------------+                 +-------------+
      / \            | claude  +-----------+ |                 | MCP Server  |
     -----        +--+ ChatGPT | MCP client+-+🕻-+           +--+ Tools:      |
    /     \       |  |   ...   +-----------+ |  |  +-----+  |  | run cmd...  |
   ---------      |  +-----------------------+  |  |     |  |  +-------------+
__/         \__   |                             +--+ MCP +--+--+ ...  
 |_    []   _|🕻---+  +-----------------------+  |  |     |  |  +-------------+
 |_   /||\  _|       |         +-----------+ |  |  +-----+  |  | MCP Server  |
 |_    /\   _|🕻------+ LLM API | MCP client+-+🕻-+           +--+ Tools:      |
---------------      |         +-----------+ |                 | fetch web...|
///////////////      +-----------------------+                 +-------------+

```

# 實際上 MCP 可視為 function calling 的 proxy

```
                              | function calling    proxy        叫用實際函式                         
       o             +--------|--------------+    （轉 接）     +-------------+
      / \            | claude |+-----------+ |                 | MCP Server  |
     -----        +--+ ChatGPT|| MCP client+-+🕻-+           +--+ Tools:      |
    /     \       |  |   ...  |+-----------+ |  |  +-----+  |  | run cmd...  |
   ---------      |  +--------|--------------+  |  |     |  |  +-------------+
__/         \__   |                             +--+ MCP +--+--+ ...  
 |_    []   _|🕻---+  +--------|--------------+  |  |     |  |  +-------------+
 |_   /||\  _|       |        |+-----------+ |  |  +-----+  |  | MCP Server  |
 |_    /\   _|🕻------+ LLM API|| MCP client+-+🕻-+           +--+ Tools:      |
---------------      |        |+-----------+ |                 | fetch web...|
///////////////      +--------|--------------+                 +-------------+
```

# 總結
## MCP 解決的事情
- 讓 LLM 可以執行實際的動作
- 讓 LLM 可以獲取外部資訊
## MCP 無關的事情
- 讓 LLM 更聰明
- 讓 LLM 自動就可以處理所有事
## MCP 的影響
- 有 API 的應用程式第一波會接上 LLM
- 接上 LLM 這件事也許可以賣錢
